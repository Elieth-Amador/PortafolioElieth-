# Portafolio Elieth

Portafolio personal desarrollado con Vite y LESS.

## Comandos

```bash
npm install
npm run dev
npm run build
```

## Entrega

En el cuadro de texto se deben colocar dos enlaces:

```text
Enlace de la página desplegada:
https://TU_USUARIO.github.io/NOMBRE_REPOSITORIO/

Enlace del repositorio:
https://github.com/TU_USUARIO/NOMBRE_REPOSITORIO
```

## Notas

- El CSS se escribe en `src/styles.less`.
- Vite genera la carpeta final `dist` con archivos minificados.
- PurgeCSS elimina reglas no utilizadas durante el build.
